# Note de fiabilité


## Description

Ce projet a pour but de donner, pour un utilisateur de Twitter donné, une note de fiabilité sur ce réseau social. Pour ce faire, on utilise le nombre de fake news qu'il publie, qu'il retweet ou qu'il like.
